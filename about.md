# Custom In-Game Progress Bar Color

Simple mod to change the progress bar color **in-game**. Also useful when using custom
(<cr>c</c><co>o</c><cy>l</c><cj>o</c><cl>r</c><cb>e</c><cp>d</c>) progress bar textures!

## Features

- **Per-context colors** for Normal, Practice, and New Best modes
- **Player colors**, custom colors, <cl>Chroma</c>, and <cp>Pastel</c> modes
- Custom gradients with **unprecedented customizability**:
  - Custom gradient colors
  - Gradient spreading
  - Animation speed
  - Light strip mode
  - ...and more!
- Compatible with <cj>Globed</c> and possibly other progress bar mods

## Things This Mod Should Not Do

- Change the color of the progress bar in other places
- Crash your game
